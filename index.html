Mampu Mengoperasikan Windows

Dibalik kesempurnaan itu Windows membutuhkan driver sebagai sarana pendukung kerja Windows tersebut. Saya sendiri sudah sekitar 7 tahun dalam mengoperasikan Sistem Operasi Windows, yg dimana berarti sudah sejak dari saya SD hingga SMK saat ini / 2015 - 2022.

Mahir Dalam Pengoperasian Adobe Illustrator

Keahlian Mengetik Keyboard Dengan Cepat Tanpa Melihat
